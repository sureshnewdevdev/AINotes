1Ô∏è‚É£ What is a Vector Database?

A vector DB stores numerical embeddings that represent meaning.

"Shiva is Dakshinamurthy."
‚Üì becomes ‚Üí
[0.11, -0.23, 0.89, ...]

Similar sentences produce similar vectors ‚Üí enabling semantic search.

7Ô∏è‚É£ Embedding Functions

from chromadb.utils.embedding_functions import SentenceTransformerEmbeddingFunction
import chromadb

client = chromadb.Client()

embedding_func = SentenceTransformerEmbeddingFunction(
    model_name="all-MiniLM-L6-v2"
)

collection = client.create_collection(
    name="shiva_collection",
    embedding_function=embedding_func
)

collection.add(
    documents=[
        "Shiva teaches yoga, music, and wisdom.",
        "Parvati is the mother goddess."
    ],
    ids=["p1", "p2"]
)

result = collection.query(
    query_texts=["Who teaches wisdom?"],
    n_results=2
)

print(result)

1Ô∏è‚É£2Ô∏è‚É£ Complete Working Project

import chromadb
from chromadb.utils.embedding_functions import SentenceTransformerEmbeddingFunction

client = chromadb.PersistentClient(path="./rag_db")

embedding_func = SentenceTransformerEmbeddingFunction(
    model_name="all-MiniLM-L6-v2"
)

collection = client.get_or_create_collection(
    name="kb",
    embedding_function=embedding_func
)

documents = [
    "Dakshinamurthy is the universal teacher.",
    "Shiva faces south in this divine form.",
    "Ganesha removes obstacles.",
    "Murugan leads the celestial army."
]

ids = ["1","2","3","4"]

collection.add(documents=documents, ids=ids)

query = "Who is the ultimate teacher?"
result = collection.query(query_texts=[query], n_results=2)

print(result)
